# CodeTech Task 4 - Sentiment Analysis using NLP

##  Objective
To perform sentiment analysis on textual data using Natural Language Processing (NLP) techniques and classify text into positive or negative sentiment.

---

## Tools & Libraries Used
- Python
- Pandas
- NLTK
- Scikit-learn

---

##  Dataset
A sample custom dataset containing text reviews labeled as:
- Positive
- Negative

---

##  Project Workflow

### 1. Data Preprocessing
- Text cleaning
- Stopwords removal

### 2. Feature Extraction
- CountVectorizer used to convert text into numerical format

### 3. Model Building
- Naive Bayes classifier (MultinomialNB)

### 4. Model Evaluation
- Accuracy score
- Classification report

### 5. Prediction
- Tested model on custom user input sentences

---

## Results
The model successfully classifies input text into positive or negative sentiment with good accuracy.

---

##  Example Predictions
- "I love this product" → Positive
- "This is very bad" → Negative



---

## 📌 Outcome
Successfully implemented an end-to-end NLP pipeline for sentiment analysis.
